# 📝 Task List - VocabForge

Berikut adalah daftar tugas (backlog) untuk pengembangan dan penyempurnaan aplikasi VocabForge ke depannya.

## 🚀 Prioritas Tinggi (Data & Security)
- [ ] **Data Export/Import**: Fitur untuk mencadangkan data (Backup) ke file JSON dan mengunggahnya kembali. Penting karena data hanya di `localStorage`.
- [ ] **Clear All Data**: Tombol reset profil dan hapus semua kata (dengan konfirmasi ganda).
- [ ] **Duplicate Check**: Validasi agar user tidak memasukkan kata yang sama dua kali.

## 🎨 Prioritas Menengah (UI/UX & Features)
- [ ] **Dark Mode**: Implementasi tema gelap menggunakan Tailwind dark mode.
- [ ] **Daily Streaks**: Sistem yang menghitung berapa hari berturut-turut user menambah kata.
- [ ] **Edit Word**: Kemampuan untuk mengoreksi kata yang sudah dimasukkan tanpa harus menghapus dan membuat baru.
- [ ] **Flashcard Mode**: Fitur belajar sederhana menggunakan kata-kata yang sudah disimpan.

## 🛠️ Prioritas Rendah (Optimization)
- [ ] **Advanced Statistics**: Grafik pertumbuhan kosakata mingguan/bulanan.
- [ ] **Sound Effects**: Menambahkan efek suara "click" taktil dan suara perayaan saat naik level.
- [ ] **Lighthouse Tuning**: Memastikan skor SEO dan Accessibility mencapai 100.
- [ ] **Custom Categories**: Pengelompokan kata berdasarkan kategori (misal: Kerja, Liburan, Hobi).

- [x] Sistem XP dan Leveling.
- [x] Sistem Lencana (Badges) dan Ruang Koleksi.

- [x] Navigasi SPA dengan View Transitions.
- [x] Fitur Pencarian Kata.
